II. The optimization problem

Logistic regression consists in finding the parameter $x \in \mathbb{R}^d$ that best fits: $$ b_i = \begin{cases} \phantom{-}1 \ \textrm{ if }\ \langle a_{i}, x\rangle \ge 0 \\ -1 \ \textrm{ if }\ \langle a_{i}, x\rangle < 0 \end{cases} $$ for all the samples $i$ of the training set (where $a_i$ is the $i^{th}$ line of $A$ and $b_i$ the $i^{th}$ element of $b$). To measure this fit, logistic regression chooses the following loss:

$$ \textrm{LogisticLoss}(b_i, \left\langle a_{i}, x\right\rangle ) = \log \left(1+\exp \left(-b_{i}\left\langle a_{i}, x\right\rangle\right)\right) $$

The corresponding optimization problem we will consider is the following:

Minimization of the empirical risk $f$ of the $L_2$-regularized logistic regression model, on the dataset $(A_{\textrm{train}},b_{\textrm{train}})$:

$$ \begin{aligned} \textrm{min}_{x\in \mathbb{R}^d} \ f(x) \ &= \ \textrm{min}_{x\in \mathbb{R}^d} \ \frac{1}{n_{\textrm{train}}} \sum_{i=1}^{n_{\textrm{train}}} f_i(x) \\ &= \ \textrm{min}_{x\in \mathbb{R}^d}\frac{1}{n_{\textrm{train}}} \sum_{i=1}^{n_{\textrm{train}}} \textrm{LogisticLoss}(b_i, \left\langle a_{i}, x\right\rangle ) + \frac{l_2}{2}\|x\|^{2}_2, \end{aligned} $$

where $l_2 \ge 0$ is the $L_2$ regularization coefficient. </div>

To simplify the notation, we'll stop indexing $A$, $b$ and $n$ with "$\textrm{train}$" in the following.

Question: Denote $\lambda_{\max}$ the largest eigenvalue of the real symmetric matrix $AA^T$.
  • Prove that $\lambda_{\max}(\nabla^2 f_i) \le \lambda_{\max}(\frac{a_i a_i^T}{4} + l_2 I_d)$
  • Prove that $\lambda_{\max}(a_i a_i^T) \le \|a_i\|^2_2$

Answer: We have $\forall i \in \{1, .., n_{train} \}$, $\forall j \in \{1, .., d \}$ : $$\frac{\partial f_i}{\partial x_j} = \frac{-b_i a_{i,j} e^{- b_i \langle a_i, x \rangle}}{1 + e^{- b_i \langle a_i, x \rangle}} + l_2 x_j$$

And $\forall i \in \{1, .., n_{train} \}$, $\forall (j,k) \in \{1, .., d \}^2$ : $$\frac{\partial^2 f_i}{\partial x_j \partial x_k} = \frac{b_i^2 a_{i,j} a_{i,k} e^{- b_i \langle a_i, x \rangle}}{(1 + e^{- b_i \langle a_i, x \rangle})^2} + l_2 \delta_{j,k} = \frac{a_{i,j} a_{i,k} e^{- b_i \langle a_i, x \rangle}}{(1 + e^{- b_i \langle a_i, x \rangle})^2} + l_2 \delta_{j,k}$$ since $b_i \in \{-1, 1 \}$.

Thus : $$\nabla^2 f_i = \frac{e^{- b_i \langle a_i, x \rangle}}{(1 + e^{- b_i \langle a_i, x \rangle})^2} a_i a_i^T + l_2 I_d$$

We know that $\lambda_{max} (\nabla^2 f_i) = max_{||x||=1} x^T \nabla^2 f_i(x) x$. We have :

$$max_{||x||=1} x^T \nabla^2 f_i x = max_{||x||=1} (x^T \frac{e^{- b_i \langle a_i, x \rangle}}{(1 + e^{- b_i \langle a_i, x \rangle})^2} a_i a_i^T x + x^T l_2 I_d x) = max_{||x||=1} (x^T \frac{e^{- b_i \langle a_i, x \rangle}}{(1 + e^{- b_i \langle a_i, x \rangle})^2} a_i a_i^T x + l_2 ||x||^2)$$

As $||x||=1$, $$max_{||x||=1} x^T \nabla^2 f_i x = max_{||x||=1} (x^T \frac{e^{- b_i \langle a_i, x \rangle}}{(1 + e^{- b_i \langle a_i, x \rangle})^2} a_i a_i^T x) + l_2$$

Also, by definition $- b_i \langle a_i, x \rangle \le 0$, so $e^{- b_i \langle a_i, x \rangle} \le 1$, and $\frac{e^{- b_i \langle a_i, x \rangle}}{(1 + e^{- b_i \langle a_i, x \rangle})^2} \le \frac{1}{4}$.

So $x^T \frac{e^{- b_i \langle a_i, x \rangle}}{(1 + e^{- b_i \langle a_i, x \rangle})^2} a_i a_i^T x \le \frac{1}{4} x^T a_i a_i^T x$ and $max_{||x||=1} x^T \nabla^2 f_i x \le max_{||x||=1} (\frac{1}{4} x^T a_i a_i^T x) + l_2 = max_{||x||=1} (\frac{1}{4} x^T a_i a_i^T x + x^T l_2 x)$.

Thus : $$\lambda_{max} (\nabla^2 f_i) \le \lambda_{max} (\frac{a_i a_i^T}{4} + l_2 I_d)$$

We have $\lambda_{max}(a_i a_i^T) \le \sum_{k=1}^d \lambda_k(a_i a_i^T) = tr(a_i a_i^T) = \sum_{k=1}^d a_k^2 = ||a_i||^2$.

Thus : $$\lambda_{max}(a_i a_i^T) \le ||a_i||^2$$ >

A consequence of these results is that the $f_i$s and $f$ are all $L$-smooth with $L \le \frac{\max_i\|a_i\|^2_2}{4} + l_2$ and we can therefore use the following value in our implementation:

L = 0.25 * max(np.linalg.norm(A,2,axis=1))**2 + self.l2

def grad(x, i=None):
    '''
    Gradient of the objective function at x.
    If no i is given, grad should return the batch gradient.
    Otherwise, it should return the partial gradient associated the the datapoint (A[i], b[i]).
    '''

    if i is None: # return batch gradient
        output = -b/(1 + np.exp(b*np.dot(A, x)))
        output = sum(np.diag(output))/n @ A
        output += self.l2*x
    else: # return partial gradient associated the datapoint (A[i], b[i])
        output = np.transpose(-b[i]/(1+np.exp(b[i]*np.dot(A[i,:], x)))*A[i,:])
        output += self.l2*x
    return output

SGD (Stochastic Gradient Descent)¶

At iteration k of the SGD algorithm, the following update is executed:

$$ \begin{cases} i \textrm{ chosen randomly in } \{1...n\}\\ x_{k+1} \leftarrow x_{k} - \gamma_k \nabla f_i(x_k) \\ \end{cases} $$
Question:
  • What stepsize $\gamma_k$ ensures the convergence of this algorithm and under what assumptions?
  • Complete TODO (5)
  • Add your code to \lib\solvers.py.

Answer: Under the assumption that f is L-smooth, $\mu$ strongly convex, and $E(||\nabla f_{i_k}(x_k)-\nabla f(x_k)||^2)<+\infty $, the algorithm converges for $\gamma_k$, s. t. $\gamma_k$ is decreasing, $\sum \gamma_k^2 < +\infty$ and $\sum \gamma_k = +\infty$. Thus, $\gamma_k = \frac{1}{k+1}$ matches.

SAGA (Stochastic Average Gradient "Amélioré")¶

SAGA starts by computing a single batch gradient, then at iteration k, the following update is executed:

$$ \begin{cases} i \textrm{ chosen randomly in } \{1...n\}\\ x_{k+1} \leftarrow x_{k} - \gamma_k\big(\nabla f_i(x_k) - \alpha_i + \bar{\alpha}\big) \\ \alpha_i \leftarrow \nabla f_i(x) \end{cases} $$

where $(\alpha_j)_{j=1}^n$ are gradients evaluated at previous iterates and $\bar{\alpha}=\frac{1}{n}\sum^n_{j=1}\alpha_j$.

SVRG (Stochastic Variance Reduced Gradient)¶

At iteration k of the SVRG algorithm, we perform the following:

$$ \begin{cases} \textrm{Compute the batch gradient } \nabla f(x_k)\\ \textrm{Initialise }y_0 = x_k\\ \textrm{for m in 0...M-1: } \begin{cases} i \textrm{ chosen randomly in } \{1...n\} \\ y_{m+1} \leftarrow y_{m} - \gamma_k\big(\nabla f_i(y_m) - \nabla f_i(x_k) + \nabla f(x_k)) \end{cases}\\ x_{k+1} \leftarrow y_{M} \end{cases} $$

def SVRG(x0, grad, max_iter, n, L, mu):
    ######## TODO (7) ########
    x = x0
    x_tab = np.copy(x)
    M = 10
    stepsize = 1./L
    for k in range(max_iter):
        ######## TODO (7) ########
        gradient = grad(x)
        y = x
        for m in range(M):
          i = np.random.randint(n)
          y = y - stepsize*(grad(y, i) - grad(x, i) + gradient)
        x = y
        x_tab = np.vstack((x_tab, x))
    return x, x_tab

Question:
  • Rewrite the empirical risk minimization objective with both $L_1$ and $L_2$ regularization (with coefficients $l_1$ and $l_2$).
  • Is this objective $\mu$-strongly convex, $L$-smooth, differentiable?
  • Complete the TODO (8) to account for the new term in the objective.
  • Add your code to lib/linear_model.py.

Answer: $$ \begin{aligned} \textrm{min}_{x\in \mathbb{R}^d} \ f(x) \ &= \ \textrm{min}_{x\in \mathbb{R}^d} \ \frac{1}{n_{\textrm{train}}} \sum_{i=1}^{n_{\textrm{train}}} f_i(x) \\ &= \ \textrm{min}_{x\in \mathbb{R}^d}\frac{1}{n_{\textrm{train}}} \sum_{i=1}^{n_{\textrm{train}}} \textrm{LogisticLoss}(b_i, \left\langle a_{i}, x\right\rangle ) + \frac{l_2}{2}\|x\|^{2}_2 + l_1 \|x\|_1, \end{aligned} $$

The $L$1 norm is not differentiable and so not $L$-smooth, but it is convex. Thus, the objective is still $\mu$-strongly convex, but not differentiable anymore, neither $L$-smooth.

self._empirical_risk = lambda x: sum(self._logistic_loss(A, b, x))/n \
                                 + self.l2/2.0 * np.linalg.norm(x,2) \
                                 + self.l1 * np.linalg.norm(x,1)

The minimization algorithms we've implemented have a slow rate of convergence for this new objective function. We will therefore use proximal variations of these algorithms.

Question: Using the $\textrm{prox}_{l_1 \|\cdot\|_1}$ operator, write down the $k^{th}$ step of proximal GD, proximal SGD, proximal SAGA, and proximal SVRG.

Answer: The $k^{th}$ step of proximal GD is :

$$x^{k+1} \leftarrow prox_{\lambda l_1 \|\cdot\|_1} (x^k - \gamma f(x_k))$$

For proximal SGD it is :

$$ \begin{cases} i \textrm{ chosen randomly in } \{1...n\}\\ x_{k+1} \leftarrow prox_{\lambda l_1 \|\cdot\|_1} (x_{k} - \gamma_k \nabla f_i(x_k)) \end{cases} $$

For proximal SAGA it is :

$$ \begin{cases} i \textrm{ chosen randomly in } \{1...n\}\\ x_{k+1} \leftarrow prox_{\lambda l_1 \|\cdot\|_1} (x_{k} - \gamma_k\big(\nabla f_i(x_k) - \alpha_i + \bar{\alpha}\big)) \\ \alpha_i \leftarrow \nabla f_i(x) \end{cases} $$

For proximal SVRG it is :

$$ \begin{cases} \textrm{Compute the batch gradient } \nabla f(x_k)\\ \textrm{Initialise }y_0 = x_k\\ \textrm{for m in 0...M-1: } \begin{cases} i \textrm{ chosen randomly in } \{1...n\} \\ y_{m+1} \leftarrow prox_{\lambda l_1 \|\cdot\|_1} (y_{m} - \gamma_k\big(\nabla f_i(y_m) - \nabla f_i(x_k) + \nabla f(x_k))) \end{cases}\\ x_{k+1} \leftarrow y_{M} \end{cases} $$
Question:
  • Give a closed-form expression of $\textrm{prox}_{l_1 \|\cdot\|_1}$
  • Complete TODO (9) (note that you can access the $l_1$ coefficient using self.l1).
  • Add your code to \lib\linear_model.py.

Answer: We have $prox_{\lambda l_1 \|\cdot\|_1} (x) = argmin_{y} (\frac{1}{2} \|y-x\|_2^2 + \lambda l_1 \|y\|_1)$.

It is equivalent to solve $argmin_{y_i} (\frac{1}{2}(y_i - x_i)^2 + \lambda l_1 |y_i|)$.

If $y_i > 0$ then the min is reached when $(y_i - x_i) + \lambda l_1 = 0$, so $y_i = x_i - \lambda l_1$ with $x_i > \lambda l_1$.

If $y_i < 0$ then the min is reached when $(y_i - x_i) - \lambda l_1 = 0$, so $y_i = x_i + \lambda l_1$ with $x_i < -\lambda l_1$.

For $x_i \in [-\lambda l_1, \lambda l_1]$, we have $y_i = 0$.

Thus : $$prox_{ \lambda l_1 \|\cdot\|_1} (x)_i = sign(x_i)max(|x_i| - \lambda l_1, 0) $$

def prox(x, stepsize):
    sign_of_x = np.where(x<0, -1, 1)
    res = np.abs(x) - stepsize*self.l1*np.ones_like(x)
    return np.multiply(sign_of_x, np.maximum(res, np.zeros_like(x)))

Convergence of gradient descent

If $f$ is $L$-smooth and $\mu$-strongly convex, for $x_0 \in \mathbb{R}^d$ and $0 < \gamma \le \frac{1}{L}$ the gradient descent converges according to : $$\|x^{k+1} - x^*\|^2_2 \le (1 - \gamma \mu)^{k+1} \|x^0 - x^*\|^2_2$$

Proof :

  • We have : $$\|x^{k+1} - x^*\|^2_2 = \|x^{k} - x^* - \gamma \nabla f(x^k) \|^2_2 = \|x^{k} - x^*\|^2_2 - 2 \gamma \langle x^{k} - x^*, \nabla f(x^k \rangle + \gamma^2 \|\nabla f(x^k)\|^2_2$$

  • Since $f$ is $\mu$-strongly convex we have $f(y) \ge f(x) + \langle \nabla f(x), y-x \rangle + \frac{\mu}{2} \|y-x\|^2_2$

So $- 2 \gamma \langle x^{k} - x^*, \nabla f(x^k \rangle \le - 2 \gamma (f(x^{k}) - f(x^*)) - 2 \gamma \frac{\mu}{2}\|x^{k} - x^*\|^2_2 $

Thus : $$\|x^{k+1} - x^*\|^2_2 \le (1 - \gamma \mu)\|x^{k} - x^*\|^2_2 - 2 \gamma (f(x^{k}) - f(x^*)) + \gamma^2 \|\nabla f(x^k)\|^2_2 $$

  • In addition, as $f$ is $L$-smooth we have $\nabla^2 f(x) \preccurlyeq L I$, so the Taylor expansion of $f$ gives :

$f(y) \le f(x) + \langle \nabla f(x), y - x \rangle + \frac{L}{2} \|y-x\|^2_2$

Thus, for $y = x - \frac{1}{L} \nabla f(x)$ we have $f(x - \frac{1}{L} \nabla f(x)) \le f(x) - \frac{1}{L} \langle \nabla f(x), \nabla f(x) \rangle + \frac{L}{2} \|\frac{1}{L} \nabla f(x)\|_2^2 $

So $f(x - \frac{1}{L} \nabla f(x)) \le f(x) - \frac{1}{2L} \| \nabla f(x)\|_2^2 $

And as $f(x^*) \le f(x - \frac{1}{L} \nabla f(x))$ by definition of $x^*$ : $ f(x^*) - f(x) \le f(x - \frac{1}{L} \nabla f(x)) - f(x) \le - \frac{1}{2L} \| \nabla f(x)\|_2^2$

Thus :

$$\| \nabla f(x)\|_2^2 \le 2L(f(x) - f(x^*))$$
  • So $\|x^{k+1} - x^*\|^2_2 \le (1 - \gamma \mu)\|x^{k} - x^*\|^2_2 - 2 \gamma (f(x^{k}) - f(x^*)) + 2L \gamma^2(f(x^k) - f(x^*)) $

Thus :

$$\|x^{k+1} - x^*\|^2_2 \le (1 - \gamma \mu)\|x^{k} - x^*\|^2_2 - 2 \gamma(1 -L \gamma)(f(x^{k}) - f(x^*)) $$
  • Since $\frac{1}{L} \ge \gamma$, we have $- 2 \gamma(1 -L \gamma) \ge 0$

So : $\|x^{k+1} - x^*\|^2_2 \le (1 - \gamma \mu)\|x^{k} - x^*\|^2_2 $

And by recurrence :

$$\|x^{k+1} - x^*\|^2_2 \le (1 - \gamma \mu)^{k+1} \|x^0 - x^*\|^2_2$$

def time_curve(*models, **named_models):
    """
    Plot the time spent by fitting to reach a certain value of empirical risk.
    The models can be passed as follows:
        learning_curve(model0, model1, name2=model2, name3=model3)
    where the `names` are display names.
    """
    plt.figure(figsize=(20,10))
    if models:
        for model in models:
            plt.plot(np.arange(0.6, 0.25, -0.05), get_time(model), linewidth=5.0, linestyle="-")

    if named_models:
        for name, model in named_models.items():
            plt.plot(np.arange(0.6, 0.25, -0.05), get_time(model), linewidth=5.0, linestyle="-", label=name)

        plt.grid(True)
        plt.legend(loc='best')
        plt.xlabel('Empirical Risk Reached')
        plt.ylabel('Computation time (in seconds)')
        plt.title('Learning curves')

    plt.show()

def get_time(model):
    """
    Get the time at which a certain value of empirical risk is reached
    """
    if not hasattr(model, "coef_"):
        raise Exception("The model hasn't been fitted yet.")
    j = 0
    x_tab = model._coef_tab
    time_tab = model._time_tab
    curve = []
    list_er = np.arange(0.6, 0.25, -0.05)
    for i, x in enumerate(x_tab):
        er = model._empirical_risk(x)
        if er<list_er[j]:
            curve.append(time_tab[i])

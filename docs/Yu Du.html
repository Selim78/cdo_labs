II. The optimization problem

Logistic regression consists in finding the parameter $x \in \mathbb{R}^d$ that best fits: $$ b_i = \begin{cases} \phantom{-}1 \ \textrm{ if }\ \langle a_{i}, x\rangle \ge 0 \\ -1 \ \textrm{ if }\ \langle a_{i}, x\rangle < 0 \end{cases} $$ for all the samples $i$ of the training set (where $a_i$ is the $i^{th}$ line of $A$ and $b_i$ the $i^{th}$ element of $b$). To measure this fit, logistic regression chooses the following loss:

$$ \textrm{LogisticLoss}(b_i, \left\langle a_{i}, x\right\rangle ) = \log \left(1+\exp \left(-b_{i}\left\langle a_{i}, x\right\rangle\right)\right) $$

The corresponding optimization problem we will consider is the following:

Minimization of the empirical risk $f$ of the $L_2$-regularized logistic regression model, on the dataset $(A_{\textrm{train}},b_{\textrm{train}})$: $$ \begin{aligned} \textrm{min}_{x\in \mathbb{R}^d} \ f(x) \ &= \ \textrm{min}_{x\in \mathbb{R}^d} \ \frac{1}{n_{\textrm{train}}} \sum_{i=1}^{n_{\textrm{train}}} f_i(x) \\ &= \ \textrm{min}_{x\in \mathbb{R}^d}\frac{1}{n_{\textrm{train}}} \sum_{i=1}^{n_{\textrm{train}}} \textrm{LogisticLoss}(b_i, \left\langle a_{i}, x\right\rangle ) + \frac{l_2}{2}\|x\|^{2}_2, \end{aligned} $$ where $l_2 \ge 0$ is the $L_2$ regularization coefficient.

To simplify the notation, we'll stop indexing $A$, $b$ and $n$ with "$\textrm{train}$" in the following.

Question: For any square matrix $M$, denote $\lambda_{\max}(M)$ the largest eighenvalue of $M$.
  • Prove that $\lambda_{\max}(\nabla^2 f_i) \le \lambda_{\max}(\frac{a_i a_i^T}{4} + l_2 I_d)$
  • Prove that $\lambda_{\max}(a_i a_i^T) \le \|a_i\|^2_2$

Answer:

  • Prove that $\lambda_{\max}(\nabla^2 f_i) \le \lambda_{\max}(\frac{a_i a_i^T}{4} + l_2 I_d)$
    • $$\nabla f_i=\frac{-b_i a_i (\exp \left(-b_{i}\left\langle a_{i}, x\right\rangle\right))}{1+\exp \left(-b_{i}\left\langle a_{i}, x\right\rangle\right)} + l_2 x$$

    Let sigmoid function $S(t) = \frac{1}{1 + e^{-t}}$

    Then $$\nabla f_i=b_i a_i(S(b_{i}\left\langle a_{i}, x\right\rangle)-1) + l_2 x$$

    $$\nabla^2 f_i=a_i S(b_{i}\left\langle a_{i}, x\right\rangle) (1-S(b_{i}\left\langle a_{i}, x\right\rangle)) a_i^T + l_2 I_d$$

    We have $$S(t) (1-S(t))=\frac{1}{e^t+\frac{1}{e^t}+2} \leq \frac{1}{4}$$

    Let $\beta = S(b_{i}\left\langle a_{i}, x\right\rangle) (1-S(b_{i}\left\langle a_{i}, x\right\rangle))$

    Thus $$\beta = S(b_{i}\left\langle a_{i}, x\right\rangle) (1-S(b_{i}\left\langle a_{i}, x\right\rangle)) \leq \frac{1}{4}$$

    $$\nabla^2 f_i=\beta a_i a_i^T + l_2 I_d$$$$(\beta a_i a_i^T + l_2 I_d) a_i = \beta a_i a_i^T a_i + l_2 a_i = \beta a_i \|a_i\|^2_2 + l_2 a_i = (\beta \|a_i\|^2_2 + l_2) a_i$$

    Thus, $\beta \|a_i\|^2_2 + l_2$ is an eigenvalue of $\nabla^2 f_i$ associated with eigenvector $a_i$.

    Let $v_k$ be another eigenvector. Since $\nabla^2 f_i$ is a symmetric matrix, $v_k$ is orthogonal to $a_i$, we have $a_i^Tv_k=0$, then:

    $$\nabla^2 f_i v_k=(\beta a_i a_i^T + l_2 I_d)v_k=\beta a_i (a_i^T v_k) + l_2 v_k=0 + l_2 v_k=l_2 v_k$$

    proving that any $v_k$ is an eigenvector of $(\nabla^2 f_i)$ associated with eigenvalue $l_2$ which is smaller than $\beta \|a_i\|^2_2 + l_2$.

    Therefore $\beta \|a_i\|^2_2 + l_2$ is the largest eigenvalue of $(\nabla^2 f_i)$ $$\lambda_{\max}(\nabla^2 f_i)=\beta \|a_i\|^2_2 + l_2$$

    In the same way, we have $$\lambda_{\max}(\frac{a_i a_i^T}{4} + l_2 I_d)=\frac{1}{4}\|a_i\|^2_2 + l_2$$ Therefore $\lambda_{\max}(\nabla^2 f_i) \le \lambda_{\max}(\frac{a_i a_i^T}{4} + l_2 I_d)$

  • Prove that $\lambda_{\max}(a_i a_i^T) \le \|a_i\|^2_2$
    • $$(a_ia_i^T)a_i=a_i(a_i^Ta_i)=a_i\|a_i\|^2_2=\|a_i\|^2_2a_i=\lambda a_i.$$

    Thus, $\|a_i\|^2_2$ is an eigenvalue of $(a_i a_i^T)$ associated with eigenvector $a_i$.

    Let $v_k$ be another eigenvector. Since $a_i a_i^T$ is a symmetric matrix, $v_k$ is orthogonal to $a_i$, we have $a_i^Tv_k=0$, then:

    $$(a_i a_i^T)v_k=a_i(a_i^Tv_k)=a_i 0=0=0v_k$$

    proving that any $v_k$ is an eigenvector of $(a_i a_i^T)$ associated with eigenvalue $0$.

    Therefore $\|a_i\|^2_2$ is the largest eigenvalue of $(a_i a_i^T)$, proving that $\lambda_{\max}(a_i a_i^T) \le \|a_i\|^2_2$.

    A consequence of these results is that the $f_i$s and $f$ are all $L$-smooth with $L \le \frac{\max_i\|a_i\|^2_2}{4} + l_2$ and we can therefore use the following value in our implementation:

    L = 0.25 * max(np.linalg.norm(A,2,axis=1))**2 + self.l2

    SAGA (Stochastic Average Gradient "Amélioré")¶

    SAGA starts by computing a single batch gradient, then at iteration k, the following update is executed:

    $$ \begin{cases} i \textrm{ chosen randomly in } \{1...n\}\\ x_{k+1} \leftarrow x_{k} - \gamma_k\big(\nabla f_i(x_k) - \alpha_i + \bar{\alpha}\big) \\ \alpha_i \leftarrow \nabla f_i(x) \end{cases} $$

    where $(\alpha_j)_{j=1}^n$ are gradients evaluated at previous iterates and $\bar{\alpha}=\frac{1}{n}\sum^n_{j=1}\alpha_j$.

    Question:
    • What stepsize $\gamma_k$ ensures the convergence of this algorithm and under what assumptions?

    • Add your code to \lib\solvers.py.

    Answer: $\gamma_k \in \left ( 0, \frac{1}{L} \right ]$ when $f$ is $\mu$-strongly convex and $L$-smooth.

    def SAGA(x0, grad, max_iter, n, L, mu):
    ######## TODO (6) ########
    stepsize  = 1 / L
    x         = x0
    x_tab     = np.copy(x)
    alpha = grad(x)
    alpha_tab = np.copy(alpha)

    for k in range(max_iter):
        ######## TODO (6) ########
        i = np.random.randint(n)
        alpha_i = grad(x, i)
        x = x - stepsize * (alpha_i - alpha + alpha_tab.mean(axis=0))
        alpha = alpha_i
        alpha_tab = np.vstack((alpha_tab, alpha))
        if k%n == 0: # each completed epoch
            x_tab = np.vstack((x_tab, x))

    return x, x_tab

    SVRG (Stochastic Variance Reduced Gradient)¶

    At iteration k of the SVRG algorithm, we perform the following:

    $$ \begin{cases} \textrm{Compute the batch gradient } \nabla f(x_k)\\ \textrm{Initialise }y_0 = x_k\\ \textrm{for m in 0...M-1: } \begin{cases} i \textrm{ chosen randomly in } \{1...n\} \\ y_{m+1} \leftarrow y_{m} - \gamma_k\big(\nabla f_i(y_m) - \nabla f_i(x_k) + \nabla f(x_k)) \end{cases}\\ x_{k+1} \leftarrow y_{M} \end{cases} $$

    def SVRG(x0, grad, max_iter, n, L, mu):
    ######## TODO (7) ########
    stepsize  = 1 / L
    x         = x0
    x_tab     = np.copy(x)
    M = 10
    for k in range(max_iter):
        ######## TODO (7) ########
        alpha = grad(x)
        y = np.copy(x)
        for m in range(M):
            i = np.random.randint(n)
            y = y - stepsize * (grad(y, i) - grad(x, i) + alpha)
        x = y

        x_tab = np.vstack((x_tab, x))

    return x, x_tab

II. The optimization problem

Logistic regression consists in finding the parameter $x \in \mathbb{R}^d$ that best fits: $$ b_i = \begin{cases} \phantom{-}1 \ \textrm{ if }\ \langle a_{i}, x\rangle \ge 0 \\ -1 \ \textrm{ if }\ \langle a_{i}, x\rangle < 0 \end{cases} $$ for all the samples $i$ of the training set (where $a_i$ is the $i^{th}$ line of $A$ and $b_i$ the $i^{th}$ element of $b$). To measure this fit, logistic regression chooses the following loss:

$$ \textrm{LogisticLoss}(b_i, \left\langle a_{i}, x\right\rangle ) = \log \left(1+\exp \left(-b_{i}\left\langle a_{i}, x\right\rangle\right)\right) $$

The corresponding optimization problem we will consider is the following:

Minimization of the empirical risk $f$ of the $L_2$-regularized logistic regression model, on the dataset $(A_{\textrm{train}},b_{\textrm{train}})$: $$ \begin{aligned} \textrm{min}_{x\in \mathbb{R}^d} \ f(x) \ &= \ \textrm{min}_{x\in \mathbb{R}^d} \ \frac{1}{n_{\textrm{train}}} \sum_{i=1}^{n_{\textrm{train}}} f_i(x) \\ &= \ \textrm{min}_{x\in \mathbb{R}^d}\frac{1}{n_{\textrm{train}}} \sum_{i=1}^{n_{\textrm{train}}} \textrm{LogisticLoss}(b_i, \left\langle a_{i}, x\right\rangle ) + \frac{l_2}{2}\|x\|^{2}_2, \end{aligned} $$ where $l_2 \ge 0$ is the $L_2$ regularization coefficient.

To simplify the notation, we'll stop indexing $A$, $b$ and $n$ with "$\textrm{train}$" in the following.

Answer:

The Logistic Loss is a composition of $x \longmapsto \log(1 + \exp{(-x)})$ and $x \longmapsto - b_i \langle a_{i}, x \rangle $. Both of these function are convex and we know that the composition of two convex functions is a convex function. So the Logistic Loss is a convex function. So it is also $0$-strongly convex, because $\textrm{LogisticLoss}(b_i, \left\langle a_{i}, x\right\rangle ) - \frac{0}{2}\|x\|^{2} $ is convex.

Furthermore, the mapping $I(x) = \frac{l_2}{2}\|x\|^{2}$ is $l_2$ strongly convex. Proof:

For $x \in R^d$ we have: $\frac{\partial I(x)}{\partial x_1} = \frac{\partial}{\partial x_1} (\frac{l_2}{2} \sum_{i=1}^d x_i ^2) = l_2 x_1 \Longrightarrow \nabla I(x) = l_2 x$.

And $I$ is $l_2$-strongly convex iff $\forall x, y \in R^d$: $$I(x) - I(y) \leq - \frac{l_2}{2} \|x - y \|^{2} + \langle l_2 x, x-y \rangle $$

$$\ \Longleftrightarrow \frac{l_2}{2} \|x\|^{2} - \frac{l_2}{2} \|y\|^{2} \leq l_2 \|x\|^{2}- l_2 \langle x, y \rangle - \frac{l_2}{2} \|x - y \|^{2} $$

$$\Longleftrightarrow \|x - y \|^{2} \leq \|x\|^{2} + \|y\|^{2} - 2 \langle x, y \rangle $$

The last equality is true. So i conclude that: $I$ is $l_2$-strongly convex.

Also, we know that if $h_1$ if $\mu_1$-strongly convex and $h_2$ is $\mu_2$-strongly convex then $(h_1 + h_2)$ is $\sigma$-strongly convex with $\sigma \geq \mu_1 + \mu_2$.

So we conclude that the function $f$ is $\mu$-strongly convex, with $\mu \geq l_2$.

So a lower bound on $\mu$ is $l_2$.

$f$ is $\mu$-strongly convex and $L$-smooth, so : $$\langle \nabla f(x) - \nabla f(y) , x - y \rangle \geq \frac{\mu L}{\mu + L} \| x - y \|^2 + \frac{1}{\mu + L} \| \nabla f(x) - \nabla f(y) \|^2 \ \ (1)$$

$\underline{Proof}$ :

Let : $g(x) = f(x) - \frac{\mu}{2} \| x \|^2$. By definition of $\mu$-strong convextity one has that $g$ is convex. Furthermore, one can show that $g$ is $(L - \mu)$ smooth by proving ( and using that it implies smoothness) the fact that if a function h is convex and $L$-smooth then : $\forall x, y \in R^n $ we have : $$ 0 \leq h(x) - h(y) - \langle \nabla h(x) , x - y \rangle \leq \frac{\beta}{2} \| x - y \|^2 $$

Thus, using the assumptions we have about $g$, one gets: $$\langle \nabla g(x) - \nabla g(y) , x - y \rangle \geq \frac{1}{L - \mu} \| \nabla g(x) - \nabla g(y) \|^2 \ \ (2)$$

In another hand, we have:

\begin{equation} \begin{split} (2) & \Longleftrightarrow \langle \nabla f(x) - \mu x - \nabla f(y) + \mu y , x - y \rangle \geq \frac{1}{L - \mu} \| \nabla f(x) - \nabla f(y) - (\mu x - \mu y) \|^2 \\ & \Longleftrightarrow \langle \nabla f(x)- \nabla f(y) , x - y \rangle - \langle \mu x - \mu y , x - y \rangle \geq \frac{1}{L - \mu} ( \| \nabla f(x) - \nabla f(y) \|^2 + \| \mu x - \mu y \|^2 - 2 \langle \nabla f(x) - \nabla f(y) , \mu x - \mu y \rangle ) \\ & \Longleftrightarrow ... \\ & \Longleftrightarrow \langle \nabla f(x) - \nabla f(y) , x - y \rangle \geq \frac{\mu L}{\mu + L} \| x - y \|^2 + \frac{1}{\mu + L} \| \nabla f(x) - \nabla f(y) \|^2 \end{split} \end{equation}

Now, using $(1)$, we get: \begin{equation} \begin{split} \| x_k - x^* \|^2 & \leq \| x_{k-1} - \gamma \nabla f(x_{k-1}) - x^* + \gamma \underbrace{\nabla f(x^*)}_{\ = \ 0} \|^2 \\ & = \| x_{k-1} - x^* \|^2 + \gamma^2 \| \nabla f(x_{k-1}) - \nabla f(x^*) \|^2 - 2 \gamma \langle \nabla f(x_{k-1}) - \nabla f(x^*) , x_{k-1} - x^* \rangle \\ & \leq (1 - \frac{2 \gamma \mu L }{L + \mu}) \| x_{k-1} - x^* \|^2 + (\gamma^2 - \frac{2 \gamma}{\mu + L} ) \| \nabla f(x_{k-1}) - \nabla f(x^*) \|^2 ) \end{split} \end{equation}

if then : $$ \| x_k - x^* \|^2 \leq (1 - \frac{2 \gamma \mu L }{L + \mu})^k \| x_{0} - x^* \|^2 $$ So : $\lim_{k \longrightarrow \infty}\| x_k - x^* \| = 0$

def GD(x0, grad, prox, max_iter, n, L, mu):
    '''
    Parameters
    ----------
    x0: array, shape (nb_features,)
        Initialisation of the solver
    grad: function
        Gradient of the objective function
    max_iter: int
        Number of iterations (i.e. number of descent steps). Note that for GD or SVRG,
        one iteration is one epoch i.e, one pass through the data, while for SGD, SAG and SAGA,
        one iteration uses only one data point.
    n: int
        Dataset size
    L: float
        Smoothness constant of the objective function
    mu: float
        Strong convexity constant of the objective function

    Returns
    -------
    x: array, shape (nb_features,)
        final iterate of the solver
    x_tab: array, shape (nb_features, max_iter)
        table of all the iterates
    '''
    stepsize = 2 / (L+mu)
    x = x0
    x_tab = np.copy(x)

    for k in range(max_iter):
        y = grad(x)
        x = x - stepsize * y
        # If l1 = 0 then the prox is the identity, i.e. prox(x) = x
        x = prox(x, stepsize)
        x_tab = np.vstack((x_tab, x))

    return x, x_tab

The Standard Gradient Descent updates the parameters only after each epoch i.e. after calculating the gradients for all the observations it updates the parameters. This phenomenon may lead to the following caveats:

  • Suppose we are working with a dataset which involves thousands of features and millions of data points (which is not uncommon in Machine Learning problems), then for each update to the parameter we would have thousand of derivatives and for each derivative, we would have millions of terms. That would be around a billion calculations for each step, which is computationally very very expensive. It will really slow down the process.

  • For large datasets, the vectorization of data doesnt fit into memory.

  • For non-convex surfaces, it may only find the local minimums.

Now let see how different variations of gradient descent can address these challenges.

SGD (Stochastic Gradient Descent)¶

In this section, we are interested in solving the following stochastic optimization problem :

$$min_{x \in R^d}(\mathbb{E}[f(x)]) \ \ \ (1)$$

To apply the SGD to the general form $(1)$ one needs to assume existence of unbiased gradient estimators. This is usually defined as follows:

$$\mathbb{E}[\nabla f_i (x)] = \nabla f(x)$$

The algorithm of this method is the following: At iteration k of the SGD algorithm, the following update is executed:

$$ \begin{cases} i \textrm{ chosen randomly in } \{1...n\}\\ x_{k+1} \leftarrow x_{k} - \gamma_k \nabla f_i(x_k) \\ \end{cases} $$

The idea of this method is that; rather than using the full gradient, we use just one training example which is super fast to compute.

Under the assumptions that, $f$ is $\mu$-strongly convex and $L$-smooth, let's check the convergence properties for this algorithm:

Let $x, y \in R^d$, we have: $$f(y) \leq f(x) + \langle y-x , \nabla f(x) \rangle + \frac{L}{2} \| x - y \|^2 $$

Let $ y = x_{k+1}$ and $x = x_k$, we have :

$$f(x_{k+1}) \leq f(x_{k}) - \gamma_k \langle \nabla f_{i}(x_k) , \nabla f(x_k) \rangle + \frac{L \gamma_k^2}{2}$$

We apply the function "Expectation" knowing $F_k$ which represents the history before $k$. we get:

\begin{equation} \begin{split} \mathbb{E}[f(x_{k+1})] & \leq f(x_{k}) - \gamma_k \langle \underbrace{\mathbb{E}[\nabla f_{i}(x_k)]}_{= \nabla f(x_k)} , \nabla f(x_k) \rangle + \frac{L \gamma_k^2}{2} \mathbb{E}[\| \nabla f_{i}(x_k) \|^2] \\ & \leq f(x_{k}) - (\gamma_k - \frac{L \gamma_k^2}{2}) \| \nabla f(x_k) \|^2 + \frac{L \gamma_k^2}{2} \underbrace{(\mathbb{E}[\| \nabla f_{i}(x_k) \|^2] - \underbrace{\| \nabla f(x_k) \|^2}_{= \|\mathbb{E}[\nabla f_{i}(x_k) \|^2} )}_{= Var(\nabla f_{i}(x_k))} \end{split} \end{equation}

With the assumption that we have a finite variance, i.e. $\exists \ \sigma > 0$ s.t. $Var(\nabla f_{i}(x_k)) \leq \sigma^2$, and by and summing over $k = 0, \ ..., \ T$ , we get that:

$$\mathbb{E}[f(x_{T+1})] + \sum_{k=0}^{T} \frac{\gamma_k (2-\gamma_k L)}{2} \mathbb{E} [ \| \nabla f(x_k) \|^2 ]\leq f(x_0) + \sum_{k=0}^{T} \frac{L \gamma_k^2 \sigma^2}{2} $$

From here, and according to what was proposed by Robbins and Monro,1951 for the classical SGD; A sufficient conditions for almost surely convergence is that:

$$\boxed{\sum_{k=0}^{\infty} \gamma_k = \infty \ , \ \sum_{k=0}^{\infty} \gamma_k^2 < \infty} $$

For example, we can take: $\boxed{\gamma_k = \frac{1}{k}}$

Furthermore, we have the following theorem: Let $f_i$ be $L$-smooth, $\sum_{k=0}^{\infty} \gamma_k = \infty$ and $\sum_{k=0}^{\infty} \gamma_k^2 < \infty$. We have:

$$\mathbb{E}[\| \nabla f(x_k) \|^2] \longrightarrow 0 \ \text{if f is convex} $$

Also, according to the work that has been showed in this article "SGD and Hogwild! Convergence Without the Bounded Gradients Assumption", we may get a better performance, if we take in addition to the last conditions, another one which is : $0 < \gamma_k \leq \frac{1}{2 L}$. Then, the following holds: $\| x_k - x^* \|^2 \longrightarrow 0$, where $x^* = argmin_{x \in R^d} f(x)$.

Finally, according to the same article, the following result establishes a sublinear convergence rate of the SGD:

Let $f$ be $\mu$-strongly convex and $L$-smooth. Let $E = \frac{2 \alpha L}{\mu}$ with $\alpha = 2$. Let's consider the SGD algorithm with a stepsize sequence such that $\gamma_t = \frac{\alpha}{\mu (t+E)} \leq \gamma_0 = \frac{1}{2 L}$. The expectation $\mathbb{E}[\| x_t - x^*\|^2]$ is at most: $\frac{4 \alpha^2 N}{\mu^2 (t-T+E)}$

For $t \geq T = \frac{4 L}{\mu}max [ \frac{L \mu}{N}\| x_0 - x^*\|^2,1 ] - \frac{4 L}{\mu}$ and $N = 2 \mathbb{E}[\| \nabla f(x^*)\|^2] $

def SGD(x0, grad, prox, max_iter, n, L, mu):
    x = x0
    x_tab = np.copy(x)

    for k in range(max_iter):
        stepsize = 1 / (k + 1)
        i = np.random.randint(n)
        y = grad(x, i)
        x = x - stepsize * y
        # If l1 = 0 then the prox is the identity, i.e. prox(x) = x
        x = prox(x, stepsize)

        if k != 0 and k % n == 0:  # each completed epoch
            x_tab = np.vstack((x_tab, x))

    return x, x_tab

The advantage of the Stochastic gradient is that each step only relies on a single gradient $\nabla f_i$, and thus the computational cost is $\frac{1}{n}$ that of the standard gradient descent. However, a disadvantage of the method is that the randomness introduces variance (this is caused by the fact that $\nabla f_i (x_k)$ is equals to $\nabla f(x_k)$ in expectation, and each $\nabla f_i (x_k)$ is different). This means that if $\nabla f_i (x_k)$ is large then we have a relatively large variance which slows down the convergence.

Also, due to the variance of random sampling, we generally need to choose $\eta_t = O(\frac{1}{t})$ and obtain a slower sub linear convergence rate of $O(\frac{1}{t})$. This is equivalent to say that we have a trade-off of fast computation per iteration and slow convergence for stochastic gradient descent versus slow computation per iteration and fast convergence for gradient descent.

In order to improve the SGD, one has to design some methods in order to reduce the variance, which allows us to use a large learning rate $\eta_t$.

SAGA (Stochastic Average Gradient "Amélioré")¶

SAGA starts by computing a single batch gradient, then at iteration k, the following update is executed:

$$ \begin{cases} i \textrm{ chosen randomly in } \{1...n\}\\ x_{k+1} \leftarrow x_{k} - \gamma_k\big(\nabla f_i(x_k) - \alpha_i + \bar{\alpha}\big) \\ \alpha_i \leftarrow \nabla f_i(x) \end{cases} $$

where $(\alpha_j)_{j=1}^n$ are gradients evaluated at previous iterates and $\bar{\alpha}=\frac{1}{n}\sum^n_{j=1}\alpha_j$.

SAGA step convergence proof¶

In this part we will demonstrate the step needed to ensure convergence for the SAGA algorithm, presented in the original paper : "SAGA, a fast incremental Gradient method with support for non-strongly convex composites objectives"

It is different from the one we used since at each step, it applies a proximal operator.

  • choose i at random uniformly
  • compute $w^{k+1}$ : $$w^{k+1} = x^k - \gamma (\nabla f_i (x_k) -\nabla f_i (x_{k-1}) + \frac{1}{n} \sum_{j=1}^n \nabla f_j (x_{k-1})) $$
  • apply the proximal operator on $w^{k+1}$ : $$ x^{k+1} = prox_{\gamma}^h (w^{k+1})$$

the proximal operator is defined as :

$$ prox_{\gamma}^h (w^{k+1}) = argmin_{x \in R^d} {h(x) + \frac{1}{2} ||x-w^{k+1}||^2} $$

To initialise the algorithm, we can perform a first step from $x_0$ to $x_1$ using a simple gradient descent, because at step 0, $x^{k-1}$ doesn't exist.

We aim to show that we achieve a convergence in the strongly convex case if $$\gamma = \frac{1}{2(\mu n +L)}$$

with L the Lipshitz constant , $\mu$ the strong convexity constant, and n ?

We present 3 Lemma used in the proof :

Lemma 1¶

with $f(x) = \frac{1}{n} \sum_{i=1}^n f_i(x)$, we suppose each $f_i$ to be $\mu-$strongly convex and to have lipschitz continuous gradient of constant L. for every $x^*$ and $x$ we then have :

\begin{align*} & <\nabla f(x), x^* -x> \leq \frac{L-\mu}{L} |f(x^*)-f(x)| \\ & - \frac{mu}{2} ||x^* - x||^2 - \frac{1}{2Ln} \sum_i ||\nabla f_i(x^*) - \nabla f_i(x) ||^2 - \frac{\mu}{L} < \nabla f(x^*),x-x^*> \\ \end{align*}

Lemma 2¶

For all x and $x^*$ we have :

convergence of SAGA theorem¶

We consider $x^*$ the optimal solution, and we define the Lyapunov function T as :

$$T^k = T(x^k,x^{k-1}) = \frac{1}{n} \sum_j f_j(x^{k-1}) - f(x^*) - \frac{1}{n} \sum_j <\nabla f_j(x^*), x^{k-1}-x^*> +c ||x^k-x^* ||^2$$

with $\gamma = \frac{1}{2 \mu n +L}$, $c= \frac{1}{2\gamma(1-\gamma \mu)n}$ and $\kappa = \frac{1}{\gamma \mu}$, we have the following conditional expectation on $T^k$ bound between steps of the SAGA algorithm :

$$ E[T^{k+1}] \leq (1-\frac{1}{\kappa}) T^k $$

Proof :

let's compute independently bounds for the three linear terms in the conditional expectation $E[T^{k+1}]$ :

$$E[ \frac{1}{n} \sum_j f_j(x^{k})] = \frac{1}{n} f(x^k) + (1-\frac{1}{n}) \frac{1}{n} \sum_j f_j(x^{k-1})$$

$$E[- \frac{1}{n} \sum_j <\nabla f_j(x^*), x^{k}-x^*>] = - \frac{1}{n} <\nabla f(x^*), x^k-x^*> - (1-\frac{1}{n})\frac{1}{n} \sum_j <\nabla f_j(x^*), x^{k-1}-x^*>$$

for the last term we apply the non-expansiveness of the proximal operator :

\begin{align*} c ||x^{k+1}-x^*||^2 & = c || prox_{\gamma} (w^{k+1}) - prox_{\gamma} (x^* - \gamma \nabla f(x^*)) ||^2 \\ & \leq c ||w^{k+1} - x^* + \gamma \nabla f(x^*)||^2 \end{align*}

Expanding the quadratic and applying $E[w^{k+1}] = x^k - \gamma \nabla f(x^k)$ we have :

\begin{align*} & c E||w^{k+1} - x^* + \gamma \nabla f(x^*)||^2 = c E || x^k - x^* + w^{k+1} -x^k +\gamma \nabla f(x^*)||^2 \\ & = c ||x^k - x^*||^2 -2 c \gamma <\nabla f(x^k) - \nabla f(x^*), x^k-x^*> + c E || w^{k+1} - x^k + \gamma \nabla f(x^*)||^2 \\ & \text{Using Lemma 3, we have :} \\ & \leq c ||x^k - x^*||^2 -2 c \gamma <\nabla f(x^k), x^k-x^*> + 2 c \gamma <\nabla f(x^*), x^k-x^*> - c \gamma^2 \beta || \nabla f(x^k) - \nabla f(x^*)||^2 \\ & + (1+\beta^{-1})c \gamma^2 E||\nabla f_i(x^{k-1})- \nabla f_i(x^*)||^2 + (1+ \beta) c \gamma^2 E|| \nabla f_i(x^k)- \nabla f(x^*)||^2 \end{align*}

now we bound $-2 c \gamma <\nabla f(x^k),x^k-x^*>$ using Lemma 1 and we use Lemma 2 to bound $E||\nabla f(x^{k-1}) - \nabla f(x^*)||^2$ :

\begin{align*} & c E||x^{k+1} - x^*||^2 \leq (c-c \gamma \mu) ||x^k-x^*||^2 +((1+\beta)c \gamma^2 - \frac{c \gamma}{L}) E||\nabla f_i(x^k) - \nabla f_i(x^*)||^2 \\ & - \frac{2 c \gamma (L-\mu)}{L} [f(x^k)-f(x^*)-<\nabla f(x^*),x^k-x^*>] - c \gamma^2 \beta || \nabla f(x^k) - \nabla f(x^*)||^2 \\ & + 2 (1+ \beta^{-1}) c \gamma^2 L [\frac{1}{n} \sum_j f_j(x^{k-1}) - f(x^*) - \frac{1}{n} \sum_j <\nabla f_j(x^*), x^{k-1}-x^*>] \\ \end{align*}

We combine all the bounds now, and using Lemma 4 we can write, for any $\kappa$ for now :

\begin{align*} & E[T^{k+1}]-T^k \leq -\frac{1}{\kappa} T^k +(\frac{1}{n} - \frac{2c \gamma (L- \mu )}{L} - 2 c \gamma^2 \mu \beta)[f(x^k)-f(x^*) - <\nabla f(x^*),x^k-x^*>] \\ & + (\frac{1}{\kappa} +2(1+ \beta^{-1}) c \gamma^2 L - \frac{1}{n}) [\frac{1}{n} \sum_j f_j(x^{k-1}) - f(x^*) - \frac{1}{n} \sum_j <\nabla f_j(x^*), x^{k-1}-x^*>] \\ & + (\frac{1}{\kappa} - \gamma \mu) c ||x^k-x^*||^2 + ((1+\beta) \gamma - \frac{1}{L}) c \gamma E||\nabla f_i(x^k)- \nabla f_i(x^*)||^2 \end{align*}

now if we pose $\boxed{\gamma = \frac{1}{2(\mu n+L)}}$, $\boxed{c = \frac{1}{2 \gamma (1-\gamma \mu ) n }}$, $\boxed{\kappa = \frac{1}{\gamma \mu}}$ and $\boxed{\beta= \frac{2 \mu n +L}{L}}$ then each quantity in the square bracket is non-positive.

We can then use the fact that $c ||x^k-x^*||^2 \leq T^k$ and by using the constant and chaining the expectation we obtain our result :

$$\boxed{E[||x^k - x^*||^2] \leq (1- \frac{\mu}{2(\mu n +L)})^k [||x^0-x^*||^2 + \frac{n}{\mu n +L} [f(x^0)-<\nabla f(x^*),x^0-x^*> - f(x^*)]]}$$

with E here being the conditional expectation over all choice of i up to step k.

def SAGA(x0, grad, prox, max_iter, n, L, mu):
    x = x0
    x_tab = np.copy(x0)
    stepsize = 1 / (2 * (mu * n + L))
    grads = np.zeros((n, len(x0)))

    for k in range(max_iter):
        i = np.random.randint(n)
        new_grad = grad(x, i)
        x = x - stepsize * (new_grad - grads[i] + np.mean(grads, axis=0))
        # If l1 = 0 then the prox is the identity, i.e. prox(x) = x
        x = prox(x, stepsize)
        grads[i] = new_grad

        if k != 0 and k % n == 0:  # each completed epoch
            x_tab = np.vstack((x_tab, x))

    return x, x_tab

SVRG (Stochastic Variance Reduced Gradient)¶

At iteration k of the SVRG algorithm, we perform the following:

$$ \begin{cases} \textrm{Compute the batch gradient } \nabla f(x_k)\\ \textrm{Initialise }y_0 = x_k\\ \textrm{for m in 0...M-1: } \begin{cases} i \textrm{ chosen randomly in } \{1...n\} \\ y_{m+1} \leftarrow y_{m} - \gamma_k\big(\nabla f_i(y_m) - \nabla f_i(x_k) + \nabla f(x_k)) \end{cases}\\ \text{Option 1}: x_{k+1} \leftarrow y_{M} \\ \text{Option 2}: x_{k+1} \leftarrow y_{t} \ \text{for randomly chosen t} \in \{0,...,M-1\} \end{cases} $$

In a practical implementation, we are going to take Option $1$, but in our analysis we are going to take Option $2$.

We will choose $M$ to be of the same order of $n$ but slightly larger (for example we will take $M=2 \times n$)

I recall that $f$ is strongly convex, $f_i$ is convex and $L$-smooth.

We have the following theorem:

  • $\underline{Theorem:}$

Let's consider the SVRG with option $2$. Under the assumption that $f_i$ is convex, smooth and that $f$ is $\mu$-strongly convex.

Let $x^* = argmin_{x} f(x)$. Assume that $M$ is sufficiently large so that:

$$\alpha = \frac{1}{\mu M \gamma(1-2 L \gamma)} + \frac{2 L \gamma}{1-2L \gamma} < 1$$

Then we have geometric convergence for SVRG:

$$\mathbb{E} P (x_{k+1}) \leq \mathbb{E} P (x^*) + \alpha^k [P(x_1) - P(x^*)] $$

  • $\underline{Proof:}$

Given any i, let: $g_i (x) = f_i (x) - f_i (x^*) - \langle \nabla f_i (x^*), x - x^* \rangle$ We have: $\nabla g_i(x) = \nabla f_i (x) - \nabla f_i (x^*) $, so $\nabla g_i (x^*) = 0$ which means that: $x^*$ is a minimiser of $g_i$. Therefore: $0=g_i (x^*) \leq min_{\gamma} [g_i (x) - \gamma \nabla f_i (x))]$ and using the fact that $f_i$ is $\mu$-strongly convex and $L$-smooth, we get that (and by taking $\gamma = \frac{1}{L}$): $0=g_i (x^*) \leq f_i (x) - \frac{1}{2 L} \| \nabla f_i (x) \|^2 $

That is: $$ \| \nabla f_i (x) - \nabla f_i (x^*) \|^2 \leq 2 L [f_i (x) - f_i (x^*) - \langle \nabla f_i (x^*), x - x^* \rangle ]$$

And by summing the above inequality over $i = 1, ..., n$, and using the fact that: $\nabla f (w^*) = 0$, we obtain:

$$\frac{1}{n} \sum_{i=1}^n \| \nabla f_i (x) - \nabla f_i (x^*) \|^2 \leq 2 L [f(x) - f(x^*)]$$

We can now proceed to prove the theorem. Let: $v_m = (\nabla f_{i_m}(y_m) - \nabla f_{i_m}(x_k) + \nabla f(x_k))$. We can take the expectation with respect to $i_m$, and obtain after some basic calculation that: $$\mathbb{E} \| v_m \|^2 \leq 4 L [ f(y_m) - f(x^*) + f(x_k) -f(x^*) ] $$

Now, by noticing that conditioned on $y_m$, we have: $\mathbb{E} v_m = \nabla f(y_m)$, and this leads to:

$$\mathbb{E} \| y_m - x^* \|^2 = \| y_{m-1} - x^* \|^2 - 2 \gamma \langle y_{m-1} - x^*, \mathbb{E} v_m \rangle + \gamma^2 \mathbb{E} \| v_m \|^2$$ So, $$\mathbb{E} \| y_m - x^* \|^2 \leq \| y_{m-1} - x^* \|^2 - 2 \gamma \langle y_{m-1} - x^*, \nabla f (y_m) \rangle + 4 L \gamma^2 [ f(y_m) - f(x^*)+ f(x_k) - f(x^*)]$$

$$\mathbb{E} \| y_m - x^* \|^2 \leq \| y_{m-1} - x^* \|^2 - 2 \gamma [f(y_m) -f(x^*)] + 4 L \gamma^2 [ f(y_m) - f(x^*)+ f(x_k) - f(x^*)] $$

$$ = \| y_{m-1} - x^* \|^2 - 2 \gamma (1 - 2 L \gamma) [f(y_m) - f(w^*)] + 4 L \gamma^2 [f(x_k) - f(x^*)] $$

The first inequality uses the previously obtained for $\mathbb{E} \| v_m \|^2$, and the second inequality convexity of $f$, which implies that: $- \langle y_m - x^*, \nabla f(y_m) \rangle \leq f(x^*) - f(y_m)$

We consider a fixed stage $k$, and $x_{k+1}$ is selected after all of the updates have completed. Bu summing the previous inequality over $t = 1,,m$ taking expectation with all the history, and using option $2$, we obtain:

$$\mathbb{E} \| y_{M} - x^* \|^2 + 2 \gamma (1-2L\gamma)m \mathbb{E} [f(x_{k+1} - f(x^*)] \leq \mathbb{E} \| y_{0} - x^* \|^2 + 4 L m \gamma^2 \mathbb{E} [f(x_{k} - f(x^*)] $$

$$ = \mathbb{E} \| x_k - x^* \|^2 + 4 L m \gamma^2 \mathbb{E} [f(x_{k} - f(x^*)] $$

$$\leq \frac{2}{\mu} \mathbb{E} [f(x_{k} - f(x^*)] + 4 L m \gamma^2 \mathbb{E} [f(x_{k} - f(x^*)]$$

$$ = 2 (\frac{1}{\gamma} + 2 L m \gamma^2 \mathbb{E} [f(x_{k} - f(x^*)])$$

The second inequality uses the strong convexity property. We thus obtain:

$$ \mathbb{E}[f(x_{k+1}) - f(x^*)] \leq [\frac{1}{\mu \gamma (1-2L \gamma)M} + \frac{2 L \gamma}{1 - 2 L \gamma}] \mathbb{E} [f(x_{k} - f(x^*)]$$

This implies that:

$$\boxed{\mathbb{E}[f(x_{k+1}) - f(x^*)] \leq \alpha^k \mathbb{E} [ f(x_1) -f(x^*)]} $$

In our procedure, we may take: $\boxed{\gamma = \frac{0.1}{L}}$ and $\boxed{M = O(n)}$ in order to obtain a convergence rate of $\alpha = \frac{1}{2}$.

Where $\gamma$ and $M$ are two constants to choose.

Answer:

The proximal operator $\textrm{prox}_{f} \ : \ R^n \longrightarrow R^n $ of a function $f$ is defined by: $$\forall v \in R^n \ \ \ \textrm{prox}_{f}(v) \ = \ argmin_{x \in R^n} \left( f(x) + \frac{1}{2}\| x - v \|_{2}^{2} \right)$$

We often use the proximal operator of the scaled function $\lambda f$, where $\lambda > 0$, which can be expressed as : $$\textrm{prox}_{\lambda f}(v) \ = \ argmin_{x \in R^n} \left( \lambda f(x) + \frac{1}{2}\| x - v \|_{2}^{2} \right)$$

We know that, if $f$ is fully separable, meaning that $f(x) = \sum_{i=1}^{n} f_{i}(x_{i})$, then:

$$(\textrm{prox}_{\lambda f}(v))_i = \textrm{prox}_{\lambda f_i}(v_i)$$

Thus, evaluating the proximal operator of a separable function reduces to evaluating the proximal operators for each of the separable parts, which can be done independently, because if we take $ (v_1,\ ... \ ,v_n) \in R^n$ we get:

\begin{equation} \begin{split} \textrm{prox}_{\lambda f}(v_1,\ ... \ ,v_n) & = argmin_{(x_1,\ ... \ ,x_n) \in R^n} \sum_{i=1}^{n} \left( \lambda f_i (x_i) + \frac{1}{2} (v_i - x_i)^2 \right) \\ & = \prod_{i=1}^{n} argmin_{x_i \in R} \left( \lambda f_i (x_i) + \frac{1}{2} (v_i - x_i)^2 \right) \\ & = \prod_{i=1}^{n} \textrm{prox}_{\lambda f_i}(v_i) \end{split} \end{equation}

Now, in one dimension, if we take $f(x) = l_1 \ |x \ |$, we know that:

$$\forall v \in R \ \ \ \textrm{prox}_{\lambda f}(v) = \textrm{prox}_{\lambda l_1 \ |x \ |}(v) = \begin{cases} v + \lambda l_1 \ \ \ \ if \ v \leq - \lambda l_1 \\ 0 \ \ \ \ \ \ \ if \ y \in [ - \lambda l_1 , \lambda l_1 ] \\ v - \lambda l_1 \ \ \ \ \ if \ v > \lambda l_1 \end{cases} $$

By generalizing to $R^n$ we get: $$ \forall v \in R^n \ \ \ \left( \textrm{prox}_{\lambda l_1 \|x \|_1}(v) \right)_i = \begin{cases} v_i + \lambda l_1 \ \ \ \ if \ v_i \leq - \lambda l_1 \\ 0 \ \ \ \ \ \ \ if \ y \in [ - \lambda l_1 , \lambda l_1 ] \\ v_i - \lambda l_1 \ \ \ \ \ if \ v_i > \lambda l_1 \end{cases} $$

$\underline{\text{A Logistic Regression Problem:}}$¶

In this section, we are going to consider the ionosphere data set.

This data set, which we denote $\mathcal{D}$ is consisting of $m = 351$ examples; each example is consisting of a feature vector $a_i \in \mathbb{R}^d$, where $d = 34$.

The goal of this supervised learning problem, which is a binary classification problem, i.e. $\mathcal{O} = \{-1 , +1 \}$ is to construct a predictor for the observations when given feature vectors. A popular approch to do so, is to use linear models, which are based on finding a parameter $w$. The latter will help us find the hyperplane wich separates well the two classes.

In order to find the parameter $w$, we use the data we have, with a loss function $l$ in order to penalizes the error between the predicted class and the true class of each example. If we take the $i^{th}$ example $(a_i , b_i)$, the corresponding error function that we take is: $f_i(x) = \ell( h(\langle a_i , w \rangle) ; b_i )$, where $h$ is the prediction function.

We want to minimize the error over all the data, so our minimization problem can be reformulated as follows:

$$\min_{w \in \mathbb{R}^d} f(w) = \min_{w \in \mathbb{R}^d} \frac{1}{m} \sum_{i=1}^m f_i(w) = \frac{1}{m} \sum_{i=1}^m \ell( h(\langle a_i , w \rangle) ; b_i )$$

We take $l$ as the Logistic loss, so the last minimization problem is equivalent to:

$$\min_{w \in \mathbb{R}^d } f(w) := \frac{1}{m} \sum_{i=1}^m \log( 1+\exp(-b_i \langle a_i,w \rangle) ) + \frac{\lambda}{2} \|w\|_2^2$$

One of the reasons why we add the last term is to avoid overfitting.

Once we find the solution of the last minimization problem, which we denote it by $w^*$, we use a bayesian approach in order to predict the class of a new example.

Indeed, if $z$ is a new example. We compute: $\mathbb{P}[z \in \text{ class } +1]$. Thus, one can decide that $z$ belongs to the class $+1$ if this probability is close to one, and to the class $-1$ otherwise.

def GD(x0, grad, prox, max_iter, n, L, mu):
    stepsize = 0.9 / L
    x = x0
    x_tab = np.copy(x)

    for k in range(max_iter):
        y = grad(x)
        x = x - stepsize * y
        x_tab = np.vstack((x_tab, x))

    return x, x_tab

In order to do better, we are going to try to use a second order algorithm, which is 'Newton Algorithm'.

The principle of this algorithm is the following:

In order to minimize a twice differentiable function $f:\mathbb{R}^n \to \mathbb{R}$, given:

  • the function to minimize f
  • a 2nd order oracle f_grad_hessian
  • an initialization point x0
  • a maximal number of iterations ITE_MAX

We do the following updates for the sequence $x_k$:

$$ x_{k+1} = x_k - [\nabla^2 f(x_k) ]^{-1} \nabla f(x_k) $$

As we see, this algorithm doesn't need to define a proper step size...

The algorithm for the second order oracle for the function $f$ defined above is the following:

def f_grad_hessian(x):
    g = np.zeros(n)
    H = np.zeros((n,n))
    for i in range(A.shape[0]):
        if b[i] > 0:
            g += -A[i]/( 1 + np.exp(np.dot( A[i] , x ) ) ) 
            H +=  (np.exp(np.dot( A[i] , x ))/( 1 + np.exp(np.dot( A[i] , x ) ) )**2)*np.outer(A[i],A[i])
        else:
            g += A[i]/( 1 + np.exp(-np.dot( A[i] , x ) ) ) 
            H +=  (np.exp(-np.dot( A[i] , x ))/( 1 + np.exp(-np.dot( A[i] , x ) ) )**2)*np.outer(A[i],A[i])
    g =  g/m + lam*x
    H = H/m + lam*np.eye(n)
    return g,H

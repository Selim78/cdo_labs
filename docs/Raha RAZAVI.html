II. The optimization problem

Logistic regression consists in finding the parameter $x \in \mathbb{R}^d$ that best fits: $$ b_i = \begin{cases} \phantom{-}1 \ \textrm{ if }\ \langle a_{i}, x\rangle \ge 0 \\ -1 \ \textrm{ if }\ \langle a_{i}, x\rangle < 0 \end{cases} $$ for all the samples $i$ of the training set (where $a_i$ is the $i^{th}$ line of $A$ and $b_i$ the $i^{th}$ element of $b$). To measure this fit, logistic regression chooses the following loss:

$$ \textrm{LogisticLoss}(b_i, \left\langle a_{i}, x\right\rangle ) = \log \left(1+\exp \left(-b_{i}\left\langle a_{i}, x\right\rangle\right)\right) $$

The corresponding optimization problem we will consider is the following:

Minimization of the empirical risk $f$ of the $L_2$-regularized logistic regression model, on the dataset $(A_{\textrm{train}},b_{\textrm{train}})$: $$ \begin{aligned} \textrm{min}_{x\in \mathbb{R}^d} \ f(x) \ &= \ \textrm{min}_{x\in \mathbb{R}^d} \ \frac{1}{n_{\textrm{train}}} \sum_{i=1}^{n_{\textrm{train}}} f_i(x) \\ &= \ \textrm{min}_{x\in \mathbb{R}^d}\frac{1}{n_{\textrm{train}}} \sum_{i=1}^{n_{\textrm{train}}} \textrm{LogisticLoss}(b_i, \left\langle a_{i}, x\right\rangle ) + \frac{l_2}{2}\|x\|^{2}_2, \end{aligned} $$ where $l_2 \ge 0$ is the $L_2$ regularization coefficient.

To simplify the notation, we'll stop indexing $A$, $b$ and $n$ with "$\textrm{train}$" in the following.

means for any i we have $∇^2𝑓_𝑖 ≤\frac{a_{ij}a_{ik}}{4}+𝑙_2𝐼_𝑑$ it is correct elementwise therefore we have the result.

For the second question:

We should asumme $a_i$ is a column of the matrix $A$ and it is $n*1$ and we have $||a_i||_2^2 = ∑_{j=1}^n a_{ji}^2$

As we know the matrix $a_ia_i^T$ will have the form

$\begin{bmatrix} a_{1i}^2 & a_{1i}a_{2i} \cdots & a_{1i}a_{ni} \\ a_{2i}a_{1i} & a_{2i}^2 \cdots & a_{2i}a_{ni} \\ \vdots & \ddots & \vdots \\ a_{ni}a_{1i} & a_{ni}a_{2i} \cdots & a_{ni}^2 \end{bmatrix}$

From the matrix $trace(a_ia_i^T)= ∑_{j=1}^n a_{ji}^2$ which is equal to $||a_i||_2^2 = ∑_{j=1}^n a_{ji}^2$

We know that $trace(a_ia_i^T)= ∑_{j=1}^m \lambda _i = \lambda _{max} +(∑_{j=1}^m \lambda _i - \lambda _{max}) = ||a_i||_2^2 $

Egienvalues of the matrix are all non-negative since $AA^T$ is symetric can be decomposed and its eigenvalues are the singular values of the matrix $A$ which are non-negative thus $(∑_{j=1}^m \lambda _i - \lambda _{max})$ is non-negative and we can get our result: $ \lambda _{max} ≤ \lambda _{max} +(∑_{j=1}^m \lambda _i - \lambda _{max}) = ||a_i||_2^2$

A consequence of these results is that the $f_i$s and $f$ are all $L$-smooth with $L \le \frac{\max_i\|a_i\|^2_2}{4} + l_2$ and we can therefore use the following value in our implementation:

GD (Gradient descent)¶

At iteration k of the GD algorithm, the following update is executed:

$$ x_{k+1} \leftarrow x_{k} - \gamma\nabla f(x_k) $$
Question:
  • What stepsize $\gamma$ ensures the convergence of this algorithm and under what assumptions?
  • Complete TODO (3) and TODO (4)
  • Add your code to \lib\solvers.py.

Answer: Suppose the function $f : R^n → R$ is convex and differentiable, and that its gradient is Lipschitz continuous with constant $L > 0$, i.e. we have that $||∇f(x) − ∇f(y)||_2 ≤ L||x − y||_2$ for any x, y. Then if we run gradient descent for k iterations with a fixed step size $t ≤ 1/L$, it will yield a solution $f^k$ which satisfies: $f(x^k) - f(x^{*}) ≤ \frac{||x^0 - x^ {*}||_2^2}{2 \gamma k}$ where $f(x^{*})$ is the optimal value. Intuitively, this means that gradient descent is guaranteed to converge and that it converges with rate $O(1/k)$.

SAGA (Stochastic Average Gradient "Amélioré")¶

SAGA starts by computing a single batch gradient, then at iteration k, the following update is executed:

$$ \begin{cases} i \textrm{ chosen randomly in } \{1...n\}\\ x_{k+1} \leftarrow x_{k} - \gamma_k\big(\nabla f_i(x_k) - \alpha_i + \bar{\alpha}\big) \\ \alpha_i \leftarrow \nabla f_i(x) \end{cases} $$

where $(\alpha_j)_{j=1}^n$ are gradients evaluated at previous iterates and $\bar{\alpha}=\frac{1}{n}\sum^n_{j=1}\alpha_j$.

import random
def SAGA(x0, grad, max_iter, n, L, mu):
    ######## TODO (6) ########
    stepsize  = 1 
    x         = x0
    x_tab     = np.copy(x)
    alpha = 0
    # choosing l randomly in {1,..., n}
    l = random.randint(0, n-1)
    alpha_l = grad(x, l)
    for  j in range(n):
        alpha += grad(x, j)
    alpha = alpha/n
    for k in range(1, max_iter + 1):
        ######## TODO (6) ########
        # choosing l randomly in {1,..., n}
        l = random.randint(0, n-1)
        stepsize = 1/k
         
        x = x - stepsize*(grad(x, l) - alpha_l + alpha)
        alpha_l = grad(x, l)
        for  j in range(n):
            alpha += grad(x , j)
        alpha = alpha/n
        if k%n == 0: # each completed epoch
            x_tab = np.vstack((x_tab, x))
    
            
    return x, x_tab

SVRG (Stochastic Variance Reduced Gradient)¶

At iteration k of the SVRG algorithm, we perform the following:

$$ \begin{cases} \textrm{Compute the batch gradient } \nabla f(x_k)\\ \textrm{Initialise }y_0 = x_k\\ \textrm{for m in 0...M-1: } \begin{cases} i \textrm{ chosen randomly in } \{1...n\} \\ y_{m+1} \leftarrow y_{m} - \gamma_k\big(\nabla f_i(y_m) - \nabla f_i(x_k) + \nabla f(x_k)) \end{cases}\\ x_{k+1} \leftarrow y_{M} \end{cases} $$

def SVRG(x0, grad, max_iter, n, L, mu):
    ######## TODO (7) ########
    stepsize  = 1 
    x         = x0
    x_tab     = np.copy(x)
    b_grad = 0
    M = 2*n
    for k in range(1, max_iter + 1):
        ######## TODO (7) ########
        stepsize = 1/k
        b_grad = grad(x, i=None)
        y = x
        for m in range(0, M-1):
            # choosing l randomly in {1,..., n}
            l = random.randint(0, n-1)
            y = y - (stepsize*(grad(y, l) - grad(x, l) + b_grad))
        x = y
        
        x_tab = np.vstack((x_tab, x))

    return x, x_tab

Answer: Minimization of the empirical risk $f$ of the $L_1$ and $L_2$-regularized logistic regression model, on the dataset $(A_{\textrm{train}},b_{\textrm{train}})$:

$$ \begin{aligned} \textrm{min}_{x\in \mathbb{R}^d} \ f(x) \ &= \ \textrm{min}_{x\in \mathbb{R}^d} \ \frac{1}{n_{\textrm{train}}} \sum_{i=1}^{n_{\textrm{train}}} f_i(x) \\ &= \ \textrm{min}_{x\in \mathbb{R}^d}\frac{1}{n_{\textrm{train}}} \sum_{i=1}^{n_{\textrm{train}}} \textrm{LogisticLoss}(b_i, \left\langle a_{i}, x\right\rangle ) + l_1\|x\|_1 + \frac{l_2}{2}\|x\|^{2}_2, \end{aligned} $$

where $l_2 \ge 0$ is the $L_2$ regularization coefficient and $l_1 \ge 0$ is the $L_1$ regularization coefficient.

It can be written that f still is a convex function since norm-1 and norm-2 are both convex. Using the definition of $\mu$-strongly convexity of a function, considering $g(x) = f(x) - \frac{l_2}{2}\|x\|^{2}_2$ is still convex so this new f is $l_2$-strongly convex. f is not differentiable everywhere since norm-1 is not differentiable at zero, except zero f is defferentiable else where. f is not differentiable everywhere so we are not able to compute the gradient and because of the norm-1 term f is not L-smooth anymore. if norm-1 be l-smooth we could have used the result of the proposition thet sumation of two l-smooth function remains l-smooth which is not the case here.

self._empirical_risk = lambda x: sum(self._logistic_loss(A, b, x))/n \

The minimization algorithms we've implemented have a slow rate of convergence for this new objective function. We will therefore use proximal variations of these algorithms.

Answer: From the definition of proximal operator the parameter $l_1$ can be interpreted as a relative weight or trade-off parameter between terms. Also $prox_{l_1||x||_1}(x)$ is a point that compromises between minimizing $f$ and being near to $x$. If $l_1$ is zero then we will get back OLS whereas very large value of $l_1$ will make coefficients zero hence it will under-fit.

def SNM(x0, grad, prox, max_iter, n, L, mu):
    #Standard Newton method
    
    epsilon  = 1/(L+mu) 
    x         = x0
    x_tab     = np.copy(x)
    for k in range(1, max_iter + 1):
        
        if np.abs(f(x)) < epsilon:
            return x, x_tab
        
        grad = grad(x, i=None)
        
        if grad == 0:                         
            print('Zero derivative. No solution found.')
            return None
        
        x = x - f(x)/grad
        
        x_tab = np.vstack((x_tab, x))

    print('No solution found.')
    return None
